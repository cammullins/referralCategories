Changing Referral Categories

Cameron Mullins
  • June 13, 2018 22:08
  • Updated
Follow

Have you run into the need to alter the client referral categories inside your YAPI software, but not sure where to start? The gif below will demonstrate the steps to change this feature, with the steps written below to assist this process.

First navigate to your YAPI dashboard:

  1. Select the Tools option from the Dashboard (suitcase icon)
  2. Select the Administrative Tools option
    1. From the subsequent drop-down, select Lists
  3. Select Referral Categories
  4. At this point, you will be given a list of referral options
    1. Select any options you would like available
    2. click Save

Finally, you will need to re-intialize your online forms:

  1. Navigate to Manage Documents and Templates
  2. Open the Public Tab
  3. From this list, select any form that lists referral categories
  4. Select Install and Initialize

These steps should yeild forms with new referral categories. If you ran into any trouble, don't hesitate to contact our tech professionals for indivudalized assitance! Just submit a request below, and our team will get back to you in no time!
